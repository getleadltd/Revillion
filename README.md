
# React Affiliate Theme

A modern WordPress theme built with React, Vite, and Tailwind CSS for affiliate marketing websites.

## Installation via Git Updater

This theme is designed to work with the [Git Updater](https://github.com/afragen/git-updater) WordPress plugin.

### Prerequisites

1. Install the Git Updater plugin in your WordPress admin
2. Ensure your WordPress site can access GitHub repositories

### Installation Steps

1. Go to your WordPress admin panel
2. Navigate to `Appearance > Themes`
3. Click `Add New`
4. Click `Upload Theme`
5. Upload this theme as a ZIP file, or use Git Updater to install directly from GitHub

### GitHub Repository Setup

Add these headers to your repository for Git Updater compatibility:
- Repository URL: `https://github.com/your-username/your-repo-name`
- Branch: `main`

### Development

To modify this theme:

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm run dev` for development
4. Run `npm run build` to build for production
5. The built files will be in the `dist/` folder

### Features

- Responsive design with Tailwind CSS
- Modern React components
- Affiliate marketing focused layout
- SEO optimized
- Fast loading with Vite bundling.
